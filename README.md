# 🦐🥚 虾蛋菜谱大全 - 可爱动漫风格菜谱管理网站

一个功能完整、可爱动漫风格的虾蛋菜谱管理网站，让您轻松记录和分享美味的虾蛋料理！

## ✨ 项目特色

- **可爱动漫设计**：温暖治愈的粉色系界面，圆润可爱的视觉元素
- **完整CRUD功能**：支持菜谱的添加、查看、编辑、删除
- **本地存储**：使用localStorage持久化存储，无需服务器
- **响应式设计**：完美适配手机、平板、桌面设备
- **图片上传**：支持菜品图片上传和预览
- **搜索排序**：实时搜索和多维度排序
- **数据导入导出**：支持JSON格式备份和恢复
- **零配置部署**：开箱即用，直接部署到GitHub Pages

## 🎯 使用指南

### 添加新菜谱
1. 点击右上角的"添加菜谱"按钮
2. 填写菜谱信息（带*为必填项）
3. 上传菜品图片（可选）
4. 添加用料清单
5. 填写详细做法
6. 点击"保存"按钮

### 管理菜谱
- **查看详情**：点击任意菜谱卡片
- **编辑菜谱**：在详情页点击"编辑"或在卡片上点击"编辑"按钮
- **删除菜谱**：在详情页点击"删除"或在卡片上点击"删除"按钮
- **搜索菜谱**：使用顶部搜索框输入关键词
- **排序菜谱**：使用排序下拉菜单选择排序方式

### 数据管理
- **导出数据**：点击右下角的"导出数据"按钮
- **导入数据**：点击"导入数据"按钮选择JSON文件
- **数据备份**：定期导出数据以防丢失

## 🐛 常见问题

### Q: 数据会丢失吗？
A: 不会！所有数据都保存在浏览器的localStorage中，除非手动清除浏览器数据。

### Q: 支持多人使用吗？
A: 目前为单用户设计，每个浏览器实例独立存储数据。

### Q: 可以部署到自己的服务器吗？
A: 可以！这是一个纯前端项目，可以部署到任何静态文件服务器。

### Q: 如何备份数据？
A: 使用"导出数据"功能将数据保存为JSON文件，需要时再用"导入数据"恢复。
